# Read FASTA File

Read DNA sequences from a FASTA file and convert to Biostrings
DNAStringSet.

## Usage

``` r
read_fasta(file_path)
```

## Arguments

- file_path:

  Character string specifying the path to the FASTA file

## Value

A DNAStringSet object containing the DNA sequences

## Examples

``` r
if (FALSE) { # \dontrun{
sequences <- read_fasta("path/to/sequences.fasta")
} # }
```
